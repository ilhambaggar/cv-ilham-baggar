📌 Description du projet

Ce projet est un site portfolio personnel développé pour présenter mon profil académique et mes compétences dans le domaine du Marketing Digital et de l’Expérience Utilisateur (UX/UI).

Le site permet de découvrir :

- mon parcours académique
- mes compétences
- mes expériences
- mes projets
- mes informations de contact

L’objectif de ce projet est de créer un CV en ligne moderne et professionnel accessible sur le web.

---

🛠️ Technologies utilisées

Les technologies utilisées pour développer ce site sont :

- HTML5
- CSS3
- Bootstrap 5
- Responsive Design

Ces technologies permettent de créer une interface moderne, responsive et facile à utiliser.
